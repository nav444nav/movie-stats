# Changelog

All notable changes to the "movie-stats" extension will be documented in this file.

## [1.0.0] - 2025-11-14

### Added
- Initial release of Movie Stats extension
- Display top 20 trending movies from TMDB
- Simple grid layout with 5 movies per page
- Pagination with Next/Prev buttons
- Activity bar integration with movie icon
- Refresh functionality
- Command palette commands
- Dark/Light theme support
- Demo API key included for immediate use
- Optional custom API key support via settings
- Comprehensive README documentation
- TypeScript implementation with strict typing
- Webpack bundling for optimized builds
- ESLint and Prettier configuration

### Features
- Clean, minimalist UI design
- Movie posters with titles and ratings
- Responsive grid layout
- Loading and error states
- XSS protection for all HTML output
- VSCode native color token usage
